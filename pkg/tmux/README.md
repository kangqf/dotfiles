每个 pkg 或 sys 目录下面包括： 
0. mapping file  // 用于或取 指定系统版本下 安装 指定场景 指定软件包版本 的命令
0. install command // 安装命令
1. .conf .rc .yml file // 配置文件目录
2. alias    // 别名目录
3. env or path config  .env .profile // 环境变量配置 
4. post install commnad // 安装后配置脚本
5. doc                 // 文档说明
6. usage or tips      // 使用说明 或 提示常见用法
7. uninstall  // 卸载命令
